# m-index-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态文件形式存放，不依赖任何后端服务，也不针对特定域名或网站。

## 项目简介

`m-index-lab` 是一个轻量级的 HTML 页面集合仓库。其主要目的是集中管理、版本化存储以及通过 GitHub Pages 等静态托管服务发布各类独立页面。这些页面可能包括个人工具页、信息展示页、实验性界面或临时归档内容。

## 目录结构

```
m-index-lab/
├── pages/           # 存放各个独立 HTML 页面及其资源
│   └── example/     # 示例子目录，每个页面可单独成文件夹
├── assets/          # 公共资源文件（CSS、JS、图片等）
├── index.html       # 仓库首页（可选，用于导航或说明）
└── README.md        # 本文件
```

- `pages/`：每个独立页面建议以文件夹形式存放，内含 `index.html` 及关联资源，便于组织与访问。
- `assets/`：存放多个页面共用的样式、脚本或图片文件，减少重复。
- `index.html`：可配置为仓库的展示首页，用于列出所有归档页面或提供简单导航。

## 页面归档说明

- 每个归档页面均为独立的静态 HTML 文件，可离线运行或直接通过浏览器打开。
- 页面内容、样式及功能由作者自行决定，本仓库仅负责归档与发布。
- 归档页面不包含任何动态后端交互，所有逻辑均在客户端完成。
- 如需添加新页面，请在 `pages/` 下创建新文件夹并放置 `index.html` 及相关资源。
